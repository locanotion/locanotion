Flock

William Bertrand, wmb
Gregory Leeper, gleeper
Nicholas Pai, npai

Flock allows users to find their friends and never get separated again. Users can either view their friend list and immediately see which eating club they are currently at, or view an attendance list at any given club. Flock uses the iPhone's built-in GPS to update each users location information dynamically and without any work on the user's end. Flock utilizes the Facebook SDK to make login and friendship information as seamless as possible for its user.
